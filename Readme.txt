#  CodSoft Data Science Internship

This repository contains machine learning projects completed as part of the **CodSoft Data Science Internship**. Each task tackles a classic data science 
problem — from predicting survival on the Titanic to detecting fraudulent credit card transactions.

##  Tasks Overview

###  Task 1 — Titanic Survival Prediction
Predict whether a passenger survived the Titanic disaster based on features such as age, gender, passenger class, and fare.

- **Type:** Binary Classification
- **Dataset:** [Titanic Dataset — Kaggle](https://www.kaggle.com/datasets/yasserh/titanic-dataset)
- **File:** https://www.kaggle.com/datasets/yasserh/titanic-dataset
- **Rename to:** Titanic-Dataset.csv
- **File:** `Task-1.ipynb`

###  Task 3 — Iris Flower Classification
Classify iris flowers into three species — *Setosa*, *Versicolor*, and *Virginica* — based on sepal and petal measurements.

- **Type:** Multi-class Classification
- **Dataset:** [IRIS Flower Dataset — Kaggle](https://www.kaggle.com/datasets/arshid/iris-flower-dataset)
- **File:** https://www.kaggle.com/datasets/arshid/iris-flower-dataset
- **Rename to:** IRIS.csv
- **File:** `Task-3.ipynb`

###  Task 5 — Credit Card Fraud Detection
Identify fraudulent credit card transactions from a highly imbalanced dataset using classification techniques.

- **Type:** Binary Classification (Imbalanced Data)
- **Dataset:** [Credit Card Fraud Detection — Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
- **File:** https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- **Rename to:** creditcard.csv
- **File:** `Task-5.ipynb`
